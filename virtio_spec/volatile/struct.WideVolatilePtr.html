Expand description

A wide volatile pointer for 64-bit fields.

In virtio, 64-bit fields are to be treated as two 32-bit fields, with low 32 bit part followed by the high 32 bit part. This type mimics VolatilePtr, and allows easy access to 64-bit fields.
