fn exit(&self, code: u32) -> !

Exit with specified return code.

Note: For X86, code is binary-OR’ed with 0x1 inside QEMU.

fn exit_success(&self) -> !

Exit QEMU using EXIT_SUCCESS, aka 0, if possible.

Note: Not possible for X86.
