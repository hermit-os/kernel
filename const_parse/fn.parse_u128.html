pub const fn parse_u128(input_string: &str) -> u128
Expand description

Parses the input into a u128 value.

  • If the input string is empty, the output is 0.
  • If the input string is exclusively decimal ASCII digits that spell a u128 value then the output will be that value.
  • Otherwise the output is an unspecified value. In this case, differing input strings are likely to produce different output values. Think of it as a very bad hash function. Use it in your roguelikes to turn the playerâ€™s name into a PRNG seed, or something like that.
