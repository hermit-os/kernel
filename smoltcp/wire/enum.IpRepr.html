pub enum IpRepr {
    Ipv4(Ipv4Repr),
    Ipv6(Ipv6Repr),
}
Expand description

An IP packet representation.

This enum abstracts the various versions of IP packets. It either contains an IPv4 or IPv6 concrete high-level representation.
